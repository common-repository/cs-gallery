Display thumbnails of photos/docs/mp3-sounds/youtubevideos and descriptions in a responsive grid and popup dialogs.

== Description ==
- Responsive thumbnail grid and popup dialog for images/document-files/mp3sounds/embedded youtube videos.
- Pop-up fullsize image for each thumbnail with description-text.
- Filter by category, place, textsearch or filetype.
- Administrator can add and edit Categories and places.
- Administrator can add and edit thumbnails and their text-descriptions.
- If categories or places filter are not wanted they can by hide in the css-file (display:none), or their names changed in the class CSGalleryLang.
- Configure the plugin by optional shortcode attributes.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add shortcode to content to display gallery.
4. Shortcode: [csgallery pagesize="10" category="" place="" type="images/documents/videos or mp3sounds" maximagepanelwidth=260 maximagepanelheight=290 language='en or se']
